# Alura Store Challenge

Análisis de datos de 4 tiendas para recomendar cuál vender.

## Objetivo
Evaluar ingresos, ventas, reseñas y costos de envío usando Python, Pandas y Matplotlib.

## Herramientas
- Python
- Pandas
- Matplotlib

## Estructura
- data/: archivos CSV
- notebooks/: análisis
- src/: funciones auxiliares
